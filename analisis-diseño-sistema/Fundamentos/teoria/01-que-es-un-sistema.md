# ¿Qué es un sistema?

## Definición

Un sistema es un conjunto de elementos relacionados entre sí que interactúan para alcanzar un objetivo común.

La naturaleza de los sistemas puede ser muy variada: desde sistemas técnicos, como aplicaciones de software y redes informáticas, hasta sistemas organizacionales, como empresas, hospitales o universidades.

---

## Explicación Feynman

Un sistema es un grupo de cosas que trabajan juntas para lograr un mismo objetivo.

Por ejemplo, una computadora no funciona únicamente gracias al procesador. También necesita memoria RAM, almacenamiento, teclado, pantalla y otros componentes. Todas estas partes colaboran para permitir que el usuario ejecute programas y realice tareas.

Si una parte importante falla, es posible que el sistema completo deje de cumplir su propósito.

---

## Características de un sistema

* Está compuesto por varios elementos.
* Sus elementos se relacionan entre sí.
* Tiene un objetivo o propósito.
* Recibe entradas y produce salidas.
* Existe dentro de un entorno.

---

## Ejemplos

### El cuerpo humano

Objetivo:

* Mantener vivo al organismo.

### Una computadora

Objetivo:

* Procesar información.

### League of Legends

Objetivo:

* Permitir una experiencia de juego competitiva.

### Una banda de metal

Objetivo:

* Crear e interpretar música.

### Una universidad

Objetivo:

* Educar y formar profesionales.

### Un hospital

Objetivo:

* Brindar atención médica.

---

## Relación con Software

Una aplicación de software también es un sistema.

Por ejemplo, un Gestor de Turnos está compuesto por:

* Usuarios
* Interfaz
* Lógica de negocio
* Base de datos

Todos estos elementos colaboran para administrar citas y servicios.
