# Gestor de Turnos

## ¿Es un sistema?

Si. 
Un sistema abierto, complejo, artificial y de tipo TPS.

## Objetivo

Gestionar y acelerar el proceso de selección de turnos para clientes.

## Componentes

Interfaz Web
API REST
Base de Datos
Clientes
Servicios
Turnos

## Entradas

Datos del cliente.
Servicio solicitado.
Fecha.
Hora.
Solicitudes de cancelación.
Solicitudes de reprogramación.

## Procesamiento

Validar que el trabajo exista y que el horario este disponible. 

## Salidas

Confirmación de turno.
Mensaje de error.
Listado de horarios.
Historial de citas.

## Retroalimentación

Cancelaciones frecuentes.
Reprogramaciones.
Opiniones de usuarios.
Errores reportados.

## Entorno

Clientes
Negocio
Internet
Horarios laborales
Empleados
Políticas de la empresa

## Clasificación

- Artificial
- Abierto
- Complejo
- Sistema de Información Transaccional (TPS)
